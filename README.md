# Calculadora con Gráficas - Proyecto Física I

Este proyecto consiste en una calculadora con interfaz gráfica desarrollada en Python para resolver y visualizar problemas de Física I.

Características
Interfaz Gráfica: Desarrollada con las librerías `tkinter, matplotlib y numpy`.
Visualización: Genera gráficas para una mejor comprensión de los fenómenos físicos.
Facilidad de uso: Diseñada para estudiantes de ingeniería.

# Requisitos
Para ejecutar este programa, necesitas tener instalado:
* [Python 3.x](https://www.python.org/)
* Librerías estándar de Python (`tkinter`).

# Instrucciones de Ejecución
1. Descarga el archivo `calcu_conejo.py` de este repositorio.
2. Abre una terminal en la carpeta donde descargaste el archivo.
3. Ejecuta el comando:
   ```bash
   python calcu_conejo.py
